其次是行内文献引用：

“设定L3下表面固定，上表面施加0.2 MPa的均匀压力载荷，计算其轴向压缩响应。计算得出单椎体轴向等效刚度约为8200 N/mm，符合健康年轻男性裸椎的高刚度特征。通过与文献\inlinecite{nekkanty2010stiffness}中对尸体进行的实验数据的比对，验证了本文逆向建模边界的准确性与材料参数设置的合理性。此步骤确立了基准模型，可进一步开展后续的双节段系统仿真。”
