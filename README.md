# 湖南大学机械与运载工程学院本科毕业设计（论文）撰写规范2025修订版（适用于2026年毕业生）

HNU机械院在2025年末发布了修订的学院毕业论文规范（适用于2026年毕业生），与校教务处发布的撰写规范有所不同。

此模板在@XayahSuSuSu [Latex-HNUThesisTemplate](https://github.com/XayahSuSuSu/Latex-HNUThesisTemplate)的基础上修改，并优化和完善了部分内容，以符合专业要求。

但是由于学院发布的撰写规范中规定的部分内容与其发布的撰写样张存在冲突与不同，需要询问院教务老师以确定最终规范要求。

目前该模板仍然处于修改阶段，以确保可以符合最终的学院要求。后续直接覆盖 `.cls` 文件即 `.sty` 文件即可。
