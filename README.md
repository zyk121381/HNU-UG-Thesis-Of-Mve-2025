# 湖南大学机械与运载工程学院本科毕业设计（论文）撰写规范（2025修订版，适用于2026届毕业生）
![Latex](https://img.shields.io/badge/Latex-XeLaTeX-61DAFB) ![License](https://img.shields.io/badge/license-Apache%202.0-green) ![Version](https://img.shields.io/github/v/release/zyk121381/HNU-UG-Thesis-Of-Mve-2025)

## 背景说明

2025年末，湖南大学机械与运载工程学院发布了修订后的本科毕业设计（论文）撰写规范，该版本专门面向**2026届毕业生**。需要特别注意的是，学院发布的规范与**学校教务处**的统一撰写规范存在差异。因此，使用本模板时请以**学院规范**为最终依据，并在有疑问时咨询学院教务老师。

## 模板来源与改动说明

本模板基于@XayahSuSuSu的[LaTeX-HNUThesisTemplate](https://github.com/XayahSuSuSu/Latex-HNUThesisTemplate) 进行二次修改，针对**机械与运载工程学院**的专业要求，进行了优化与完善，修正了原模板中与学院规范不符的细节问题。

## 已知问题：规范与样张存在冲突

目前，学院发布的**规范文档**与其提供的**撰写样张**之间存在部分内容不一致的情况，例如：

- 目录格式与字体要求；
- 章标题编号样式；
- 各类排版样式要求；
- ……

**处理建议：**  2026年4月8日接机械院教务处老师通知，按照学院提供的样张模板来。

## 当前模板状态

本模板仍处于**修订阶段**，主要进行各模块定位距离的微调。  
**当前版本**可用于日常撰写与排版练习，但建议在正式提交前，确认最终规范是否有进一步调整。

## 更新方式

本项目的更新主要通过覆盖以下文件完成：  
- `hnu.cls`（核心格式控制文件）  
- 相关 `.sty` 文件（宏包配置）  

后续只需下载最新版本，将上述文件替换到项目目录中，重新编译（使用`XeLatex`）即可。

## 使用建议

1. 仔细阅读学院发布的《2025修订版撰写规范》和样张；
2. 使用本模板进行排版时，建议开启 `draft` 模式先行检查内容；
3. 遇到格式问题，优先对比学院样张，而非直接参照学校模板；
4. 如发现本模板与学院最终规范存在出入，欢迎提 Issue 或联系维护者。
